# CIS300-S26-Coppick
SnapVault – Secure Project Management Platform

SnapVault is your all-in-one solution for storing, managing, and sharing projects securely. Built with modern web technologies, SnapVault combines sleek design with practical functionality to keep your projects organized, accessible, and collaborative.

🌟 Features

Secure Storage: Keep projects safe with organized document management.

Lightning Fast Access: Instant upload and download speeds across devices.

Team Collaboration: Share projects with team members and work in real time.

Responsive Design: Works seamlessly on desktops, tablets, and mobile devices.

Interactive Navigation: Easy-to-use nav menu with search functionality.

Short Video Demo: Includes a built-in <video> showing how SnapVault works.

Forms & Links:

Contact email link for support.

External link for additional resources.

Project upload form (4+ fields) for adding new projects.

Custom CSS Styling: Vibrant, modern design using gradients, shadows, and hover effects.

🛠️ Technologies Used

Languages: HTML5, CSS3, JavaScript

Media: <video> tag, responsive images

Design: Flexbox & Grid (for medium/large screens), responsive layout

Browser Features: Fully functional search bar with live feedback

⚡ How to Use

Open index.html in your browser.

Navigate using the top navigation bar: Home, Projects, Project Upload, About.

Use the search bar to quickly find a project (currently uses dummy data).

Watch the demo video to see SnapVault in action.

Click “Get Started” or “View Projects” buttons to interact with content.
