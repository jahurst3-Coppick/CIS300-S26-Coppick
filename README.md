# CIS300-S26-Coppick

## SnapVault – Secure Project Management Platform
🔗 **Live Site:** https://jahurst3-coppick.github.io/CIS300-S26-Coppick/

SnapVault is a modern web application designed for storing, managing, and sharing projects securely. It combines a clean, responsive design with practical functionality to keep projects organized, accessible, and easy to collaborate on.

---

## 🌟 Features

- **Secure Storage**  
  Organize and manage projects using structured local storage and UI-based project tracking.

- **Lightning-Fast Access**  
  Quickly upload, retrieve, and view project data across pages.

- **Team Collaboration (Simulated)**  
  Project structure supports sharing and multi-user expansion in future development.

- **Responsive Design**  
  Fully optimized for desktop, tablet, and mobile devices.

- **Interactive Navigation & Search**  
  Navigation bar includes a live search system that filters stored projects.

- **Embedded Video Demo**  
  HTML5 video element demonstrates platform functionality.

---

## 📁 Project Management System

- Create new projects via upload form  
- Edit project details inline  
- Delete projects from table view  
- Persist data using `localStorage`  
- Track project status (active, completed, urgent)  
- Store images with captions per project  

---

## ✨ New Interactive Features

- **🎉 Confetti Celebration**
  - Confetti animation triggers when a new project is successfully created
  - Enhances user feedback and engagement

- **🛠️ Animated 404 Page**
  - Custom “Page Not Found” experience
  - Includes visual icons and styled layout for better UX

- **📋 Copy Email with Toast Notification**
  - Click-to-copy email functionality
  - Visual confirmation via toast message

- **🖼️ Drag & Drop Image Upload**
  - Users can drag and drop images into the upload zone
  - Instant preview with editable captions and removal options

## ⚡ How to Use

1. Open `index.html` or visit the live site.
2. Navigate using the top menu:
   - Home
   - Projects
   - New Project
   - About
3. Create a project using the upload form.
4. Add images via drag & drop or file selection.
5. Submit to trigger a confirmation and confetti animation.
6. View, edit, or delete projects on the Projects page.
7. Use the search bar to locate saved projects.
